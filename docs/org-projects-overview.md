## Finding Your Own Contributions

Use the bundled script to filter the table above to only repositories where a
specific GitHub user has made commits:

```bash
# Install dependency (curl + python3 required)
chmod +x scripts/org-contributor-report.sh

# List all org projects (no filter)
./scripts/org-contributor-report.sh

# List only projects where <your-github-username> has commits
./scripts/org-contributor-report.sh <your-github-username>

# Same, using a personal access token to avoid GitHub API rate limits
GITHUB_TOKEN=ghp_xxxx ./scripts/org-contributor-report.sh <your-github-username>
```

The script outputs a formatted table and a summary of languages used.
